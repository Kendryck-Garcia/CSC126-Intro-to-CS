# CSC 126 - Introduction to Computer Science
Repository of all lab and homework assignments completed at CSI covering fundamental C++ programming concepts.

## Course Overview
This repository contains 23 C++ programming assignments that progressively cover core computer science concepts, from basic I/O and control structures to more advanced topics like arrays, functions, and file handling.

## Topics Covered

### 1. **Basic I/O & Syntax**
- Console input/output (cin, cout)
- String handling and string libraries
- Program structure and comments

### 2. **Selection & Control Flow**
- If/Else statements for conditional execution
- Nested conditionals for complex decision-making
- Multi-branch selections (if-else-if chains)
- Switch-like functionality with string comparisons

### 3. **Repetition & Loops**
- While loops for conditional iteration
- For loops with loop counters
- Nested loops for multi-dimensional problems
- Loop control and incrementing patterns

### 4. **Modular Programming (Functions)**
- Function declarations and definitions
- Parameter passing and return types
- Function calls and reusability
- Helper functions for code organization

### 5. **Data Types & Variables**
- Primitive data types (int, double, float)
- String variables and manipulation
- Array declarations and initialization
- 2D arrays for matrix-like data

### 6. **Data Structures**
- Single-dimensional arrays for collections
- 2D arrays for tabular data
- Parallel arrays for related data
- Struct concepts through organized data

### 7. **Mathematical Operations**
- Arithmetic operations and expressions
- Mathematical functions (sqrt, pow, etc.)
- Formatting output with setw and setprecision
- Calculations and formulas

### 8. **Input Validation & Error Handling**
- User input validation
- Handling invalid inputs gracefully
- Try-catch concepts (basic error management)
- Logic validation

### 9. **File I/O**
- Reading from files
- Writing to files
- File stream operations

## Learning Progression
1. **Foundation**: Basic syntax, variables, and output
2. **Logic**: Conditional statements and decision-making
3. **Iteration**: Loop structures and repetition
4. **Modularity**: Breaking programs into functions
5. **Data Organization**: Arrays and structured data
6. **Integration**: Combining concepts in complex programs
